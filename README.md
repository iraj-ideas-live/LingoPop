# LingoPop

یک اپ ساده وب برای مدیریت لغت، سینک با Google API و تمرین با جعبه‌های G5.

## اجرا

- فایل `index.html` را در مرورگر باز کنید.
- برای موبایل هم کافی است همان فایل را در مرورگر موبایل باز کنید.

## JSON ورودی

فرمت فایل می‌تواند این باشد:

```json
["apple", "book", "travel"]
```

یا:

```json
[
  { "word": "apple" },
  { "word": "book" }
]
```

## نکته امنیتی

کلید API در مرورگر ذخیره می‌شود (LocalStorage). برای محیط‌های عمومی توصیه نمی‌شود.
